Installation

Installation der veröffentlichten Version

Kopiere das Archiv ZZZ_courseplay.zip in deinen Mods-Ordner. Das war's!

Der Mods-Ordner befindet sich:

  • Windows (7 und höher):
  • C:\Users\[BENUTZERNAME]\My Documents\MyGames\FarmingSimulator2015\mods
  • Mac OS X (Download/DVD/Steam-Version):
  • /Users/[BENUTZERNAME]/Library/Application Support/FarmingSimulator2015
  • Mac OS X (X 10.7 und höher):
  • /Users/[BENUTZERNAME]/Library/Containers/com.focus-home.farmingsim2015/Data/Library/Application Support/FarmingSimulator2015

Installation der Entwickler-Version

Entpacke die heruntergeladene Zip-Datei in einen Ordner deiner Wahl. Hier findest du nun einen neuen Ordner namens courseplay-master. Packe den Inhalt dieses Ordners in eine neue Zip-Datei names ZZZ_courseplay.zip. Dann installiere diese normal wie oben beschrieben.
